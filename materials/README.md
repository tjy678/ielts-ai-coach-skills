# 备考资料文件夹

将你的雅思备考资料文件放入对应子目录，Skills 将自动读取并在训练中引用。

## 目录结构

```
materials/
├── README.md           # 本说明
├── cambridge/          # 剑桥真题系列
├── writing/            # 写作范文、模板、题目集
├── reading/            # 阅读文章、题目、同义替换表
├── listening/          # 听力音频、transcript、场景词汇
├── speaking/           # 口语话题卡、范例回答、Part 3 题库
├── vocab/              # 词汇表、AWL 学术词汇、话题词汇
└── templates/          # 答题模板、笔记模板、模考答题卡
```

## 支持的文件格式

- **文本**：`.md` `.txt` `.json` `.csv`
- **文档**：`.pdf` `.docx`
- **图片**：`.png` `.jpg` `.webp`（题目截图、答题卡等）
- **表格**：`.xlsx`（词汇表、成绩记录等）
- **音频**：`.mp3` `.wav` `.m4a`（听力材料）

## 使用方式

1. 将资料文件拖入对应文件夹
2. 在对话中告诉 AI：「用我的资料」或「查看资料文件夹」
3. Skills 会自动扫描并引用相关资料

## 各文件夹说明

### cambridge/ — 剑桥真题
- 剑桥雅思真题 PDF
- 答案和解析
- 分数换算表

### writing/ — 写作资料
- 高分范文
- Task 1 常用句型模板
- Task 2 话题论点库
- 常见写作题目汇总

### reading/ — 阅读资料
- 阅读真题文章
- 同义替换对照表
- 题型分类练习

### listening/ — 听力资料
- 听力 transcript
- Section 场景词汇
- 高频拼写词汇表
- 地图题方位词

### speaking/ — 口语资料
- Part 2 话题卡
- Part 1/3 常见问题
- 口语范例回答

### vocab/ — 词汇资料
- AWL 学术词汇表
- 话题分类词汇
- 同义替换表

### templates/ — 模板资料
- 作文结构模板
- 口语笔记模板
- 模考答题卡
- 学习计划模板
