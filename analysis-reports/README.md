# 模考分析报告

此目录存放每次模考后的智能解读报告，由 ielts-analysis 模块自动生成。

## 文件命名规则

```
exam-{cambridge_number}-test{test_number}-{date}.md
```

## 报告内容

每份报告包含：
1. 整体概览（各科正确数/Band/错题数）
2. 听力逐题分析（按 Section，含原文定位 + 错因分类 + 正确答案）
3. 阅读逐题分析（按 Passage，含题型 + 解析 + 错因分类）
4. 能力短板诊断（按 Section/Passage 统计弱点）
5. 针对性提分建议（精确到题型和训练量）

> 个人模考报告不会被纳入版本控制（.gitignore）。
